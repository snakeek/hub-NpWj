# Task2：判断“你”字位置的五分类任务

## 1. 任务说明

输入一个任意包含“你”字的五个字文本。

“你”在第几位，就属于第几类。

例如：

| 输入文本 | “你”的位置 | 类别 |
|---|---:|---:|
| 你今天好 | 第 1 位 | 第 1 类 |
| 我你今天 | 第 2 位 | 第 2 类 |
| 今天你好 | 第 3 位 | 第 3 类 |
| 今天我你 | 第 4 位 | 第 4 类 |
| 今天好你 | 第 5 位 | 第 5 类 |

在代码内部，类别编号使用 `0~4` 进行训练；预测展示时转换为 `1~5`。

## 4. 训练模型

```bash
python train.py
```

默认会自动随机生成训练数据，不需要手工准备数据集。

训练数据规则：

1. 文本长度固定为 5；
2. 文本中一定包含“你”；
3. “你”出现在哪个位置，标签就是哪个类别。

训练完成后，模型会保存到：

```text
outputs/task2_model.pt
```

---

## 5. 预测示例

```bash
python predict.py --text "今天你好"
```

输出示例：


再试一个：

```bash
python predict.py --text "我很想你"
```

## 6. 可调训练参数

```bash
python train.py \
  --train_size 5000 \
  --val_size 1000 \
  --epochs 5 \
  --batch_size 64 \
  --lr 0.001
```

参数说明：

| 参数 | 说明 | 默认值 |
|---|---|---:|
| `--train_size` | 随机生成的训练样本数 | 5000 |
| `--val_size` | 随机生成的验证样本数 | 1000 |
| `--epochs` | 训练轮数 | 5 |
| `--batch_size` | 批大小 | 64 |
| `--lr` | 学习率 | 0.001 |
| `--output_dir` | 模型保存目录 | outputs |

## 7. 模型结构

这是一个非常简单的字符级分类模型：

```text
5 字文本
  ↓
字符转 ID
  ↓
Embedding
  ↓
Flatten
  ↓
Linear + ReLU
  ↓
Linear
  ↓
5 类分类
```

这个任务本身非常简单，本质上是学习一个位置判断规则。理论上不用深度学习也可以直接写规则完成：

```python
label = text.index("你") + 1
```

但这里为了跑通“文本输入 → 模型训练 → 模型保存 → 模型预测”的完整流程，所以仍然使用了一个小型神经网络。
