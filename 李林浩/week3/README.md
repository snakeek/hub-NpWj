# 第三周作业：文本分类任务实验

本周作业主要完成两个文本分类实验，分别对应 `task1` 和 `task2` 两个目录。

- `task1`：设计一个以文本为输入的多分类任务，实验使用 RNN、LSTM、GRU 等模型完成训练。
- `task2`：对一个任意包含“你”字的五个字文本进行分类，“你”在第几位，就属于第几类。
